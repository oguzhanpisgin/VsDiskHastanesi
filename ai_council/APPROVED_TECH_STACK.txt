========================================
RATIONALE
========================================
1. Vue.js 3 chosen over Alpine.js:
   - Complex CMS/CRM dashboards need robust framework
   - TypeScript support for type safety
   - Better component reusability
   - Easier migration to advanced features

2. Container Apps chosen:
   - Better scalability than App Service
   - Microservices-ready architecture
   - Lower cost than AKS
   - Kubernetes-compatible (future AKS migration)

3. Vertical Slice Architecture:
   - Feature independence
   - Faster development cycles
   - Perfect for modular system (CMS/CRM/Website/Chatbot)
   - Easier to maintain than Clean Architecture

4. Azure AD B2C:
   - Microsoft-managed security
   - Built-in MFA and conditional access
   - Social logins (Google, Microsoft, Apple)
   - No per-user fees (vs IdentityServer hosting costs)
